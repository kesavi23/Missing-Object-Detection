# 🔍 Missing Object Detection System

An AI-powered Missing Object Detection System that helps users locate and monitor everyday objects using **YOLO**, **OpenCV**, **Speech Recognition**, and **Voice Alerts**.

The application provides real-time object detection, voice-controlled interaction, object tracking, and missing object alerts through an interactive desktop interface built with Python.

---

## 🚀 Features

- 🎯 Real-time object detection using YOLO
- 🎤 Voice command support
- 🔊 Text-to-Speech voice feedback
- 📦 Missing object tracking and alerts
- 📍 Last seen object location detection
- 📈 Detection heatmap visualization
- 📸 Snapshot capture
- 📄 PDF report generation
- 🖥️ Interactive Tkinter GUI
- 📹 Camera switching support
- 🧠 Scene memory for object tracking
- 🗂️ Detection logging using SQLite
- 🤖 Custom object training support
- ⚡ GPU acceleration (CUDA) when available

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Missing-Object-Detection.git

cd Missing-Object-Detection
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python obj.py
```

---

## 🎙️ Voice Commands

| Command | Action |
|----------|--------|
| Start | Start camera |
| Stop | Stop camera |
| Find Bottle | Track bottle |
| Find Phone | Locate phone |
| What do you see | Describe detected objects |
| Snapshot | Save current frame |
| Export PDF | Generate report |
| Summary | Detection statistics |
| Camera One | Switch camera |
| Mute | Disable voice |
| Unmute | Enable voice |
| Quit | Exit application |
